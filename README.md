# UniGames - Social Board Game Hub

A unified social board gaming platform where players can create custom rooms, lounge together, and play real-time board games.

## Features
- **Auth System**: Fully persistence-mocked JWT authentication.
- **Account Badges**: Custom roles for Admin, Moderator, Founder, Contributor, and Verified users.
- **Onboarding Flow**: Step-by-step setup wizard for avatar selection, usernames, categories, and tags.
- **Session Management**: Audit connection histories and revoke active device sessions.
- **Settings Dashboard**: Adjust profile bio, change credentials, switch theme modes (Dark/Light/Gaming), and configure notification/privacy settings.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
